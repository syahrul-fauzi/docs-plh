## 1. Pelaporan Deployment

Setiap deployment ke environment staging atau production harus didokumentasikan
dengan langkah-langkah berikut:

### Langkah-langkah

1. **Salin Laporan**: Seluruh anggota tim wajib menyalin file laporan deployment
   ke dalam sistem ticketing internal.
2. **Lokasi File**: File laporan asli disimpan di `docs/deployment/reports/`.
3. **Verifikasi Akses**: Pastikan file telah terupload ke repository.
4. **Deadline**: Pelaporan harus diselesaikan sebelum **EOD (End of Day)**.
