# User Manual - Lawyers Hub

Selamat datang di Lawyers Hub! Panduan ini akan membantu Anda memahami cara menggunakan platform kami untuk mengelola kasus hukum, dokumen, dan kolaborasi tim secara efisien.

## 1. Memulai (Getting Started)

### Login dan Autentikasi
- Buka [https://app.lawyershub.com](https://app.lawyershub.com)
- Masuk menggunakan akun profesional Anda (Clerk Auth).
- Jika Anda baru, pastikan Anda telah menerima undangan dari admin firma Anda.

## 2. Manajemen Kasus (Case Management)

### Membuat Kasus Baru
1. Klik menu **"Cases"** di sidebar kiri.
2. Klik tombol **"+ New Case"**.
3. Isi detail kasus (Nama, Tipe Hukum, Klien, Deskripsi).
4. Klik **"Create"**.

> ðŸ“¸ **Screenshot**: [Formulir Kasus Baru](https://link-to-screenshot.com/new-case-form.png)

### Melacak Status Kasus
Setiap kasus memiliki status (Draft, Active, Pending, Closed). Anda dapat memperbarui status melalui halaman detail kasus.

## 3. Manajemen Dokumen & Editor Kolaboratif

### Mengunggah Dokumen
- Di dalam detail kasus, buka tab **"Documents"**.
- Drag & Drop file atau klik **"Upload"**.

> ðŸ“½ï¸ **Video Tutorial**: [Manajemen Dokumen & Kolaborasi](https://link-to-video-tutorial.com/documents)

### Editor Kolaboratif (Real-time)
- Klik pada dokumen tipe `.law` atau buat dokumen baru.
- Editor kami mendukung kolaborasi real-time (mirip Google Docs).
- Perubahan tersimpan secara otomatis.

## 4. Fitur AI & Compliance

### Audit PII (Personal Identifiable Information)
Sistem secara otomatis mendeteksi informasi sensitif (Nama, NIK, Alamat) dalam dokumen untuk memastikan kepatuhan terhadap UU PDP.

### AI Legal Assistant
Gunakan Copilot di sisi kanan layar untuk:
- Merangkum dokumen hukum panjang.
- Memeriksa klausul yang berisiko.
- Memberikan saran terminologi hukum.

## 5. Troubleshooting & FAQ

**Q: Saya tidak bisa mengunggah dokumen.**
*A: Pastikan format file didukung (PDF, DOCX, LAW) dan ukuran file tidak melebihi 20MB.*

**Q: Bagaimana cara menambah anggota tim?**
*A: Hanya Admin yang dapat menambah anggota melalui menu **"Settings > Members"**.*

**Q: Lupa password?**
*A: Gunakan fitur "Forgot Password" pada halaman login Clerk.*
