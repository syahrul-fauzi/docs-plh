## 1. Pelaporan Deployment
Setiap deployment ke environment staging atau production harus didokumentasikan dengan langkah-langkah berikut:

### Langkah-langkah:
1. **Salin Laporan**: Seluruh anggota tim yang terlibat dalam deployment wajib menyalin file laporan deployment (misal: `LH-DEPLOY-20260104-001.md`) ke dalam sistem ticketing internal.
2. **Lokasi File**: File laporan asli disimpan di direktori `docs/deployment/reports/`.
3. **Verifikasi Akses**: Pastikan file telah terupload ke repository dan dapat diakses oleh semua stakeholder terkait (Tech Lead, QA, Compliance).
4. **Deadline**: Pelaporan harus diselesaikan sebelum **EOD (End of Day)** pada hari deployment dilakukan.
